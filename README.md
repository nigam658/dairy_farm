# dairy_farm
This is a beginner-friendly dairy farm web application built using Flask. Technologies used:  Python (Flask) MySQL HTML &amp; CSS

  Features--
  User Signup & Login system
  Secure password hashing
  Session-based authentication
  Protected homepage (only accessible after login)
  Logout functionality
  MySQL database integration



  Tech Stack
  Backend: Python (Flask)
  Database: MySQL
  Frontend: HTML, CSS
  Security: Werkzeug (Password Hashing)
  Environment Management: python-dotenv

 
  Security Note

Sensitive data like database password and secret key are stored in .env file and are not included in the repository.


Purpose

This project is built for learning Flask and backend development.
It demonstrates how to build a basic authentication system with database connectivity.
